# Smart Table Agent：设计系统与 AI 的深度融合

## 1. 项目愿景：让设计系统"活"起来
我们正在构建一个**懂设计系统的 AI 助手**。它不仅仅是一个自动化脚本，而是一个能理解设计师意图、熟悉设计规范的“虚拟搭档”。通过对话，它能瞬间搭建出符合高保真标准的表格界面。

## 2. 核心价值：从“手动搬砖”到“意图设计”
*   **懂你所想 (Intent-Driven)**：你只需要说“我要一个包含状态和操作列的用户表”，或者直接丢一张截图，AI 就能识别出你的意图，自动匹配最合适的组件。
*   **懂设计规范 (System-Aware)**：
    *   **自动组件化**：AI 生成的不是死板的图层，而是**标准的组件实例**。
    *   **智能适配**：它知道“操作列”需要自适应宽度，知道“状态标签”应该用哪个变体。它像一位熟练的设计师，自动处理了所有繁琐的规范细节。
*   **所见即所得 (Seamless Interaction)**：
    *   **无缝迭代**：觉得列宽不对？想换个表头样式？直接告诉 AI，它会精准修改，就像你在指导一位助手改图，而不是重新生成。

## 3. 创新模式：AI Agent (智能体)
我们将 AI 升级为了一个**有手有眼的智能体 (Agent)**：
*   **眼 (感知)**：它能看懂当前选中的表格结构，理解设计稿的上下文。
*   **脑 (决策)**：它基于我们的设计规范进行思考，决定用什么组件、设什么参数。
*   **手 (执行)**：它直接操作 Figma，精准完成从布局到填充的全流程。

## 4. 总结
Smart Table Agent 不仅提升了设计效率，更重要的是**降低了设计系统的使用门槛**。它让每一位设计师都能通过简单的自然语言，轻松调用最规范的设计资产，释放精力专注于更有价值的创造性工作。
